# Data-Metabase

This project sets up a Metabase instance with MariaDB using Docker Compose.

## Prerequisites

- Docker
- Docker Compose

## Getting Started

1. Clone the repository.
2. Set up your environment variables in a `.env` file (refer to `.env.example` if available).
3. Run `docker-compose up -d`.

## Services

- **Metabase**: Business intelligence tool.
- **MariaDB**: Relational database.
